UserDefaultsStorage

public struct UserDefaultsStorage : Storage

A struct holding reference to all Secrets stored in the UserDefaults.

Warning

UserDefaults are not safe for storing Secrets. DO NOT USE THIS IN PRODUCTION.

Note

KeychainStorage is the encoded and ready-to-use alternative to UserDefaultsStorage. Add https://github.com/evgenyneu/keychain-swift to your dependencies and import it to start using it.
